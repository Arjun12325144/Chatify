This is basically the chating website , i named it "Chatify" and the technologies it involves is React for UI part, nodeJs and Express for backend part and MongoDb as database.
I also integrate Socket.io library for real time messaging and check whether the user is online or offline.

checkout -> https://chatify-ortso.sevalla.app/
